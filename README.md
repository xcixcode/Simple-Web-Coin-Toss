# Simple-Web-Coin-Toss

**사용자가 직접 파워를 조절하여 던지는 재미를 느낄 수 있도록 제작한 인터랙티브 웹 동전 던지기**

외부 리소스(이미지, 사운드 파일) 없이 순수 웹 기술(Vanilla JS)로만 구현되어 가볍고 빠르게 작동하는 3D 동전 던지기 시뮬레이터입니다.  
사용자가 버튼을 누르는 시간에 따라 파워가 결정되며, 정밀한 사운드 합성 로직과 CSS 3D 시퀀스를 통해 몰입감 있는 사용자 경험을 제공합니다.

---

## 개요

**Simple-Web-Coin-Toss** 프로젝트는 브라우저 내에서 물리적인 상호작용과 소리를 얼마나 정교하게 구현할 수 있는지 탐구한 프로젝트입니다.  
`setTimeout`과 `transitionend` 이벤트를 유기적으로 연결하여 동전의 도약, 체공, 낙하 단계를 제어하며, 모든 시각적 요소와 소리는 코드를 통해 실시간으로 생성됩니다. 단일 파일 구조로 설계되어 별도의 서버나 라이브러리 없이도 완벽한 기능을 보장합니다.

---

## 기술 스택

- **HTML5 / CSS3** - `perspective`와 `preserve-3d`를 활용한 3D 가상 공간 및 동전의 앞/뒷면 구조 설계  
- **JavaScript (Vanilla JS)** - 비즈니스 로직 제어 및 `setInterval`을 이용한 실시간 파워 게이지 시스템 구축  
- **Web Audio API** - `OscillatorNode`와 `GainNode`를 제어하여 'toss(휘두르는 소리)'와 'drop(금속 타격음)' 사운드 합성  
- **CSS Transitions** - `cubic-bezier` 곡선을 이용한 물리적 낙하 가속도 구현 및 애니메이션 상태 관리  

---

## 주요 기능

- **인터랙티브 파워 게이지**: `mousedown/touchstart` 유지 시간에 따라 0~100 사이를 왕복하는 동적 게이지 시스템 (3.8단위 가중치 적용)
- **정밀한 사운드 동기화**: 
    - **Toss**: 300Hz에서 1200Hz로 급격히 상승하는 주파수를 통해 공기를 가르는 소리 구현
    - **Drop**: 800Hz, 1100Hz, 1500Hz의 다중 주파수 합성을 통해 실감 나는 금속성 충돌음 구현
- **CSS 3D 애니메이션 시퀀스**:
    - **상승 단계**: `scale(0.7)`과 `translateY`를 조합하여 원근감 있는 도약 연출
    - **낙하 단계**: `isLandingPhase` 플래그와 `transitionend` 이벤트를 결합하여 지면에 닿는 순간의 정확한 사운드 피드백 제공
- **시각적 UX 장치**: 결과 도출 전 `filter: blur(25px)`를 적용한 **프라이버시 블러** 모드로 긴장감 조성
- **반응형 아케이드 UI**: `viewport-fit=cover`와 `dvh` 단위를 사용하여 모바일 상단 바와 노치를 고려한 풀스크린 레이아웃 제공

### 로컬에서 실행
1. `index.html` 파일을 다운로드하거나 코드를 복사합니다.

3. 외부 라이브러리 의존성이 없어 인터넷 연결 없이도 모든 핵심 로직이 작동합니다.
